Impact of Storms and Severe Weather Events on Public Health and Economy consequences in the United States

Synonpsis

In this report, we aim to analyze the impact of different weather events on public health and economy based on the storm database collected from the U.S. National Oceanic and Atmospheric Administrations (NOAA) . We will use the estimates of fatalities, injuries, property and crop damage to decide which types of event are most harmful to the population health and economy. From these data, we found that excessive heat and tornado are most harmful with respect to population health, while flood, drought have the greatest economic consequences across the United States

Analysis of Weather Events on Public Health

In this code chunk we take top 10 events that are responsible for causing severe damage on public health. In our data we have FATALITIES and INJURIES are variables that are recorded the public health damage, using these variables we find the events that are dangerous.

agg1 <- aggregate(INJURIES~EVTYPE,FUN = sum,data = storm_data)
agg2 <- aggregate(FATALITIES~EVTYPE,FUN = sum, data = storm_data)
evinj <- filter(agg1, INJURIES != 0)
evfat <- filter(agg2, FATALITIES != 0)
fatalities <- arrange(evfat, desc(FATALITIES))[1:10,]
injuries <- arrange(evinj, desc(INJURIES))[1:10,]

Analysis of Weather Events on Economy

The data storm_data consists the variables CROPDMGEXP and PROPDMGEXP describes the multiplier to CROPDMG and PROPDMG which is described in (storm documentation). That is Hundreds(H),Thousands(K),Millions(M) and Billions(B). Example : If we have 5 in CROPDMG and M in CROPDMGEXP then the total is 5 millions. Like this we filter the data and takes the top 15 events affects most to economy.

prop_dmg <- filter(select(storm_data,EVTYPE,PROPDMG,PROPDMGEXP),PROPDMG != 0)
crop_dmg <- filter(select(storm_data,EVTYPE,CROPDMG,CROPDMGEXP),CROPDMG != 0)
datahelper <- function(dataset){
dataset <- filter(dataset, dataset[,3] == "B" |dataset[,3] == "M"|dataset[,3] == "K"|dataset[,3] == "H" )
dataset[,3][toupper(dataset[,3]) == "B"] <- 10^9  
dataset[,3][toupper(dataset[,3])== "M"] <- 10^6  
dataset[,3][toupper(dataset[,3]) == "K"] <- 1000  
dataset[,3][toupper(dataset[,3]) == "H"] <- 100
dataset[,3][toupper(dataset[,3]) == ""] <- 0  
dataset$total <- dataset[,2] * as.numeric(dataset[,3])
return(dataset)
}
property <- datahelper(prop_dmg)
crop <- datahelper(crop_dmg)
propagg <- aggregate(total ~ EVTYPE,data = property,FUN = sum)
cropagg <- aggregate(total ~ EVTYPE,data = crop,FUN = sum)
evprop <-arrange(propagg, desc(total))[1:15,]
evcrop <- arrange(cropagg,desc(total))[1:15,]
